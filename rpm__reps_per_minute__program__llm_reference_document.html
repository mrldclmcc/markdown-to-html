RPM (Reps Per Minute) Program: LLM Reference Document

This document provides a comprehensive overview of the Brian Alsruhe's Reps Per Minute (RPM) program, designed to enable an LLM to generate customized workouts and full programs based on user requests.

1. Program Overview

The RPM program is a 9-week, full-body strength and conditioning program specifically designed to be run in a caloric deficit. Its primary goals are to burn body fat, build work capacity, and enhance technical proficiency. Unlike programs focused solely on maximizing strength, RPM aims to make the user leaner, more athletic, and explosive.

It is structured to be less physically demanding on the central nervous system (CNS) and spine than heavier programs, making it an excellent "bridge program" between more intense training cycles. It is also highly suitable for individuals with less time for the gym as each session is efficient and time-bound.

The core principle is "Reps Per Minute" (RPM), where every second in the program is "on the clock," making the clock a "brutal coach".

2. Key Principles and Mechanics

2.1. "Every Minute on the Minute" (EMOM) Style

The vast majority of this program is run in an "Every Minute on the Minute" (EMOM) style.

  • The user will need a timer or clock to watch.
  • At the start of every new minute, the user begins their set for a prescribed number of repetitions and then rests for the remainder of that minute.
  • Faster completion of reps means more rest time within the minute.
  • The clock creates a sense of urgency and forces the user to "answer the bell" at the start of each new minute, promoting mental toughness and discouraging quitting. This repetitive decision-making (to persevere or quit) is highlighted as a habit-forming process that can bleed into other areas of life.

2.2. Full-Body Training

  • The RPM program is the creator's first full-body program.
  • Each training session covers the entire body.
  • This approach is specifically chosen for the program's goals, as it allows for better work capacity development and provides a CNS and spine break from constant heavy loading.
  • It also means that if a day is missed, it's "not a big deal" because the entire body is worked multiple times a week.

2.3. Intensity and Percentage Guidelines (HLM - Heavy, Light, Medium)

The program utilizes an undulating intensity throughout the week and within each daily session. This means the user is "never going to be completely punishing" themselves on any single day.

Intensities are categorized as Heavy, Light, or Medium (HLM) and are based on percentages of the user's 1 Repetition Maximum (1RM). The percentages and corresponding rep ranges increase across the three 3-week waves.

Here's how HLM intensities translate to percentages and rep ranges per wave:

  • Wave 1 (Weeks 1-3):

    • Light: 40% of 1RM for 5-6 reps per minute
    • Medium: 50% of 1RM for 4-5 reps per minute
    • Heavy: 60% of 1RM for 3-4 reps per minute

  • Wave 2 (Weeks 4-6):

    • Light: 45% of 1RM for 5-6 reps per minute
    • Medium: 55% of 1RM for 4-5 reps per minute
    • Heavy: 65% of 1RM for 3-4 reps per minute

  • Wave 3 (Weeks 7-9):

    • Light: 50% of 1RM for 5-6 reps per minute
    • Medium: 60% of 1RM for 4-5 reps per minute
    • Heavy: 70% of 1RM for 2-3 reps per minute (Note: The source text for Wave 3 percentages has a slight discrepancy with the actual table provided, where it states "70% you're only looking at two three reps" while the table shows 70% for 2-3 reps for Heavy. The table values are more consistent with the progression for the Heavy category.)

  • Note on Percentage Progression: Each wave generally increases the percentage by 5% for each intensity level, while the rep ranges for Light, Medium, and Heavy largely remain consistent with the initial guidelines (except for the Heavy category in Wave 3). The maximum intensity reached is around 70-75% over the entire program.

2.4. Autoregulation

The program incorporates autoregulation, allowing users to adjust the workout based on their feeling and performance.

  • The rep ranges (e.g., 5-6 reps) provide flexibility: if feeling good, aim for the higher end; if struggling, stick to the lower end.
  • If a set feels too easy, users can increase the weight within the 10-minute block or add extra core or conditioning exercises after completing the main mover reps for that minute. This drastically changes things by increasing workload and decreasing rest.
  • If a set feels too heavy or the user is running out of time, they should drop the reps back or decrease the weight.

2.5. Emphasis on Form and Bar Speed

  • The program emphasizes technical proficiency and explosiveness.
  • Users are instructed to "Never sacrifice FORM for SPEED and never sacrifice SPEED for WEIGHT".
  • The goal is to ingrain good reps, ensuring "every rep you perform is a good one".
  • The lighter nature of the program allows for intense focus on technique without excessive worry about injury.

2.6. Variation Utilization

  • The program includes a variety of lifts and variations to address common weaknesses in main movers and keep the program engaging.
  • Users are given the option to choose variations or stick to conventional lifts, especially if they are powerlifters.
  • Determining Weight for Variations without a 1RM: If a user doesn't know their 1RM for a specific variation (e.g., Floor Press), they should use the 10-minute block in a "ramping fashion".
    • Start with a lighter weight and gradually add weight each minute as long as good form is maintained and bar speed doesn't slow unacceptably.
    • Once an appropriate weight is found, the user can stay with it for the remainder of the sets or continue to manipulate it based on energy levels.
    • After a week or two, the user should have a good idea of their 1RM for that variation.

2.7. Recovery

  • While 24-48 hours between barbell sessions is encouraged, RPM is described as the least physically demanding of the creator's programs.
  • Recovery is "slightly less of a priority," allowing for harder active recovery, supplementary sandbag sessions, or side sports without significant negative effects.

3. Program Structure Details

The RPM program is 9 weeks long, divided into three 3-week waves. It is performed 4 times per week, with each session consisting of 40 minutes of actual work.

Each daily session is broken down into four 10-minute blocks, totaling 40 minutes of work. The amount of time spent warming up or "messing around" will determine the total gym time, but it can easily be done in 50-60 minutes.

The four 10-minute blocks typically cover:

  1. Lower Body Focus: Deadlifts, Squats, or variations
  2. Back Focus: Rows, Pull-ups, Carries, or Loads
  3. Press Focus (Upper Body): Overhead Press (vertical) or Bench Press (horizontal) or variations
  4. Core or Conditioning Focus: Abdominal work, conditioning exercises, or a combination

The intensities (Heavy, Light, Medium) for these focus areas undulate throughout the week to ensure varied stimulus and recovery. For example, if deadlifts are heavy on Day 1, squats might be light on Day 2, and the next time deadlifts appear in the week (e.g., Day 3), their intensity will be different (e.g., medium).

4. Target Audience and Use Cases

The RPM program is designed for:

  • Individuals looking to lose body fat and get leaner.
  • Those aiming to build work capacity and improve conditioning.
  • Athletes wanting to enhance technical proficiency and bar speed in their lifts.
  • People seeking a "bridge program" to recover their CNS and spine after an intense, heavy training cycle, before transitioning to another heavy program.
  • Individuals with limited time for gym sessions who need efficient, impactful workouts.
  • Those who want to be bigger, faster, stronger, and more conditioned without necessarily pursuing world-champion powerlifting or strongman goals. It's about overall well-rounded fitness, "superhero type of training".
  • Users interested in exploring and finding PRs for various lifting variations in a fun and engaging way.

It is NOT for: individuals specifically aiming to add huge amounts to their 1RM (e.g., taking a 700lb deadlift to 750lb), as its focus is broader than pure maximal strength.

5. Important Considerations for Customization

When generating RPM customized workouts or programs, the LLM should adhere to these principles:

  • Adherence to EMOM Structure: Every set must be "Every Minute on the Minute" for 10 minutes per block.
  • Intensity Undulation: Ensure that the HLM (Heavy, Light, Medium) intensities are varied across the week and within each day, as specified in the program structure (e.g., if a Deadlift is Heavy on Day 1, the Squat on Day 2 should be Light, and the next Deadlift variation on Day 3 would be Medium).
  • Correct Percentage and Rep Range Application: Use the HLM percentages and rep ranges corresponding to the correct wave (Wave 1: 40-50-60%, Wave 2: 45-55-65%, Wave 3: 50-60-70% with adjusted reps for heavy).
  • Exercise Selection:
    • Each day must include a Lower Body, Back, and Press (Vertical or Horizontal) focus.
    • Odd days typically end with Core/Assistance, while even days end with Conditioning.
    • When suggesting variations, explain that they target common weaknesses or offer more fun, and how to auto-regulate the weight if a 1RM is unknown (ramping).
    • If suggesting specific exercises, ensure they align with the program's examples (e.g., for "Load Focus," suggest a Sandbag Carry or a Stone Load, or offer a squat/row alternative if equipment is an issue).
  • Autoregulation Options: Always include the option for the user to add extra core/conditioning exercises after the main mover within the minute if the prescribed work feels too easy. Emphasize that these additions should be done after the main lift.
  • Form Over Weight/Speed: Remind the user about the importance of good form and maintaining bar speed, never sacrificing either for weight or speed.
  • Mental Aspect Integration: Encourage perseverance and emphasize the "answer the bell" mentality.
